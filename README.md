# TensorFlow-for-DeepLearning
## Python 환경

- anaconda 4.6.11
  - python 3.6
  - numpy 1.17.3
  - matplotlib 3.1.1
  - tensorflow 2.0.0

[Python 환경 구축 가이드](https://github.com/ParkJH1/Python-Tutorial/blob/master/README.md)



## 환경 설정

1. conda 환경 생성 및 활성화

```
>>> conda create -n python-env python=3.6
>>> conda activate python-env
```



2. conda 환경 목록

```
>>> conda env list
```



3. python 모듈 설치

```
>>> pip install -r requirements.txt
```
